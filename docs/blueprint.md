# **App Name**: Radiant Canvas

## Core Features:

- Service Portfolio Display: Showcase a beautiful portfolio of makeup services with professional images, detailed descriptions, and pricing.
- Direct WhatsApp Booking: Enable clients to easily initiate booking requests for specific services via WhatsApp with pre-filled message templates.
- Admin Access Toggle: Implement a discreet, hidden trigger for the artist to enter or exit an administrative mode for managing services.
- Service Management Portal (Admin): Allow the makeup artist to create, edit, and delete service listings, including name, description, and pricing, stored securely in a database.
- Image Uploader (Admin): Provide a tool for the artist to upload and update captivating images for each makeup service listing.
- AI Style Inspiration Tool (Admin): Offer an AI-powered tool to generate creative names or descriptive text suggestions for new makeup looks and services.

## Style Guidelines:

- Primary color: Elegant deep rose (#993355), embodying sophistication and warmth for key elements and accents.
- Background color: Soft, desaturated rose (#F5EFEB), providing a clean and inviting canvas that harmonizes with the primary color.
- Accent color: Vibrant, analogous purple-pink (#CA5EAF), used sparingly for interactive elements and highlights to create visual interest.
- Headline font: 'Playfair' (serif), chosen for its high-end, elegant aesthetic, suitable for service titles and headers.
- Body text font: 'PT Sans' (humanist sans-serif), offering excellent readability and a modern, friendly feel for descriptions and general information.
- Utilize a minimalist and elegant icon set (e.g., Lucide Icons) that maintains a consistent line weight and style to complement the clean design.
- A responsive, modern grid-based layout for presenting services, ensuring optimal viewing across various device sizes. Clear, spacious presentation for individual service details.
- Subtle hover effects and transition animations on service cards and interactive elements, providing a polished and engaging user experience.
